# Published Meta-Analyses: Colchicine for Cardiovascular Prevention

Validation reference for CardioSynth Phase 0.

## Key Validation Targets

CardioSynth pooled estimates should fall within these ranges:
- **MACE: RR/HR 0.73-0.88** (broad range across 15 published MAs)
- **MI: RR 0.74-0.85**
- **Stroke: RR 0.67-0.90**
- **All-cause mortality: ~1.00 (no effect)**
- **CV mortality: ~0.94 (NS)**
- **GI AEs: RR ~1.68 (significant harm signal)**

Note: Post-CLEAR SYNERGY (2025, n=7,062, NEUTRAL) estimates are attenuated vs pre-2025.

## Gold Standard: Ebrahimi (2025) Cochrane Review

- k=12, N=22,983, PMID: 41224205
- MI: **RR 0.74 (0.57-0.96); I2=51% [HIGH-CERTAINTY]**
- Stroke: **RR 0.67 (0.47-0.95); I2=40% [HIGH-CERTAINTY]**
- All-cause mortality: RR 1.01 (0.84-1.21); I2=1%
- CV mortality: RR 0.94 (0.73-1.22); I2=13%
- GI AEs: RR 1.68 (1.11-2.57); I2=91%

## Key Trials in Published MAs

- **LoDoCo (2013)**: n=532, stable CAD, colchicine 0.5mg/d
- **COLCOT (2019, NCT02551094)**: n=4,745, recent MI, HR 0.77 for MACE
- **COPS (2020)**: n=795, ACS — neutral primary, harm signal
- **LoDoCo2 (2020, NCT02278471)**: n=5,522, stable CAD, HR 0.69 for MACE
- **CONVINCE (2024)**: n=3,144, post-stroke
- **CLEAR SYNERGY (2025)**: n=7,062, post-STEMI — NEUTRAL (attenuates pooled estimates)

## CardioSynth Validation Notes

CardioSynth uses CT.gov structured results ONLY. Key differences from published MAs:
1. Only trials that POSTED results to CT.gov are included
2. Effect sizes calculated from structured fields, not full-text extraction
3. Published MAs may include trials not on CT.gov (e.g., ISRCTN registry)
4. Our k will likely be smaller than published MAs (fewer trials have posted structured results)
5. Expect pooled RR for MACE in 0.65-0.90 range if COLCOT + other posting trials included

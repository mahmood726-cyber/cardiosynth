CardioSynth

Living Cardiology Evidence Synthesis Engine

Browser-native · zero-dependency · provenance-tracked · ClinicalTrials.gov structured results only

How it works. Each tool is a single self-contained HTML file that runs entirely in your browser with no server, build step, or external dependency. Numbers pass fail-closed TruthCert validation; pooling uses DerSimonian–Laird random effects with HKSJ correction, validated against R metafor.
